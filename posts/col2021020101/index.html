经验记录, Hugo - LoveIt 主题初始使用的时候一些小问题.

1 安装.

1.1 基本过程

我使用的环境是mac 和 hugo 0.8 具体如何安装可以在网上很容易找到, (后面再写详细一点的)

1. hugo new site mysite
2. cd mysite 
3. git clone  https://github.com/dillonzq/LoveIt.git  themes/loveit
4. cp -r themes/LoveIt/exampleSite/content ./
5. 因为 作者应该可以正常访问 tw 和 ins ,在大陆里面访问不到,
所以会出现错误. 所以要去修改. 
具体文件路径:

…./mysite/themes/LoveIt/exampleSite/content/posts/theme-documentation-built-in-shortcodes/index.en.md index.fr index.zh-cn 这三个文件里面的内容都需要修改, 包括已经发布的文件里面的index 里面,总共是6个文件. .

2 发布流程.

2.1. 发布文章

修改完这个,然后去在 content/posts 下面建立文件目录 wenjian1, 建立后,在里面建立 index.md 开始正式写文章. 建立了一个sample 目录,这样每次只要copy 这个目录并改名,后面再去 进行编辑就好了,其它的都不用动了.

2.2 一个注意

inddex.md 这个文章不能为空, 是内容不能为空,一定要含一些能显示的内容,就是在 — 下面正文中要有内容,不然编译要出错

2.3 插入日期

文章发布的时候,我是在VSCode 之中进行编辑, index.md 文件模版需要填入一串日期. 这个很麻烦,最好是使用 vscode 中的一更扩展
这个是要设置一下日期参数,不然文章里面读不出来, 其实就是在日子和小时中加入一个T字母

2.4 插入图片

如果是截图相对还是比较容易的,使用一个插件就好了
可以一键完成任务.

2.5 时间问题

当前的插入用的是北京时间 和 标准时间差8个小时,不能发布
